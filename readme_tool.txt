English
==========================
* Outline
Thank you for downloading Effekseer.
In this package only the tool for creating effects for designers is saved.

When playing in the game, please download other runtime, Unity, DX library etc package.


* Operating environment
- Windows Vista or later PC with a video card compatible with ShaderModel 2.0


*Required runtime
You need to install the following runtime.

- Microsoft .NET Framework 4 (Full)
Please download from the following address.
https://www.microsoft.com/en-US/download/details.aspx?id=17851

- The latest DirectX end user runtime
Please download from the following address.
https://www.microsoft.com/en-US/download/details.aspx?id=35


* Directory structure
-Tool
Tools for creating effects are saved.

-Sample
Sample effects provided by various users are saved.

-Help
Help that describes how to use the tool is saved.


* For those using Effekseer for the first time

Let's first download the .NET Framework and the DirectX end user runtime to satisfy the operating environment.

Let's start Effekseer.exe in the Tool directory.
I think that you do not know the meaning at first when a lot of parameters exist, but let's push the playback button (triangle mark) at the bottom left for the time being.
Then a square was displayed on the screen. This is the simplest effect.

Let's open the effect file in the Sample directory with Effekseer as it is boring as it is.
The effect of Effekseer has an extension of .efkproj. I will open this with Effekseer.
Then you can see various effects.

Click on the node tree node in the upper right corner to see the parameters of the node.
Let's change the numerical value of sample effect variously. Even just that, you can make effects that are used for many games.

Let's create an effect by yourself as soon as you get used to fingering samples.
You can try creating effects from your past experiences, and you can practice by watching the tutorial in Help.

As long as you use the sample effect with Effekseer, you can use it freely, so you can use the remodeled effects directly in your game without problems.
For 2D games, you can output it as an image from the recording window, and you can easily play it even during the game with the DX library or Unity.

■　初めてEffekseerを使用する方へ

まずは動作環境を満たすように.NET FrameworkやDirectX エンド ユーザー ランタイムをダウンロードしましょう。

そして、Toolディレクトリ内のEffekseer.exeを起動してみましょう。
沢山パラメーターが存在して最初は意味がわからないと思いますが、とりあえず左下の再生ボタン(三角のマーク)を押してみましょう。
そうすると、画面に四角が表示されました。これは一番シンプルなエフェクトになります。

このままではつまらないので、Sampleディレクトリ内のエフェクトファイルをEffekseerで開いてみましょう。
Effekseerのエフェクトは、.efkprojという拡張子がついています。これをEffekseerで開いてみます。
そうすると様々なエフェクトを見ることができます。

右上のノードツリーのノードをクリックすると、ノードのパラメーターを見ることができます。
サンプルエフェクトの数値を色々変えてみましょう。それだけでも多くのゲームに使用されるようなエフェクトは作れます。

サンプルを弄るのに慣れたら、いよいよ自分でエフェクトを作成してみましょう。
今までの経験からエフェクトを作成してみるのもいいですし、Help内にあるチュートリアルを見て練習しても問題ありません。

サンプルのエフェクトはEffekseerで使用する限り、自由に使えるので改造したエフェクトをそのまま自分のゲームで使用しても問題ありません。
2Dゲームなら、録画ウインドウから画像として出力してもいいですし、DXライブラリやUnityなら簡単にゲーム中でも再生できます。

* 개요
 Effekseer를 다운받아줘서 고맙습니다.
이 패키지 안에는 디자이너들을 위해 효과를 만들수 있는 툴만 저장되어있습니다.

D이 게임을 플레이할 때, 다른 런타임이나 유니티또는 DX 라이브러리 etc 패키지를 다운받아야합니다.

* 권장 
- 윈도우 비스타 그 이후 PC, 그래픽 카드는 쉐이더 2.0이상 지원하는 모델


*요구되는 런타임
다음과 같은 런타임이 됩니다.

- 마이크로소프트 .NET 프레임워크 4 (Full)
아래 URL로 다운받으면 됩니다.
https://www.microsoft.com/en-US/download/details.aspx?id=17851

- 가장 최근의 DirectX 런타임
아래 URL로 다운받으면 됩니다.
https://www.microsoft.com/en-US/download/details.aspx?id=35


* 디렉토리 
-툴
 툴은 이펙트를 생성하고 저장할 수 있습니다.

-샘플
샘플 효과는 다른 유저들이 만든 것을 보여줍니다.

-도움
도움은 툴의 사용과 저장에 대해 설명해줍니다.


* 처음 Effekseer를 사용하는 사람들을 위한 설명

처음에 .Net 프레임워크와 DirectX 최신 버전을 운영체제에 맞게 다운로드합니다.

이제 Effekseer.exe를 툴 디렉토리 안에서 실행합니다.
내 생각에 첫 실행시 파라미터 예외처리 오류가 발생한다면, Playback 버튼(삼각형 모양)을 한동안 눌러야합니다.
그리고나서 스크린에 네모가 뜹니다. 그게 가장 심플한 효과입니다.

샘플 디렉토리에서 Effekseer를 지루하게 하는 효과 파일을 엽니다.
Effekseer효과의 확장자는 .efkproj입니다. 이 확장자는 Effekseer에서 열 수 있습니다.
그러면 당신은 다양한 효과들을 볼 수 있습니다.

노드 트리 오른쪽 코너 위에 파라미터 노드가 보이면 눌러줍니다.
이제 심플한 효과의 수치를 다양하게 바꿔봅시다. 이를 통해 당신은 많은 게임에서 사용되는 효과를 만들 수 있습니다.

이제 샘플을 만지는데 능숙해짐에 따라 당신 스스로 효과를 만들어봅니다.
당신은 과거의 경험을 통해 효과를 만들 수 있고, 당신은 도움을 통해 튜토리얼로 보고 연습할 수 있습니다.

Effekseer의 샘플을 사용하는 한, 당신은 무료로 문제없이 리모델링한 효과를 사용할 수 있습니다.
2D게임의 경우 레코딩 창을 통해서 출력이 가능하고, DX라이브러리나 유니티 사용시에도 쉽게 게임을 플레이할 수 있습니다.
