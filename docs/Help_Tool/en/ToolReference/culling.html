Culling

Overview

Set parameters related to culling. Culling is set so that effects are not drawn outside the screen. If the rough shape set by culling is not drawn outside the screen, the effect itself is not drawn as well. When culling is not set, the effect is always drawn.

Parameter

カリングの表示

Set whether to display the culling shape on the screen.

カリングの種類

Set the type of culling shape.

なし

It does not set culling. The effect is always drawn.

球

It is the shape of a sphere.

半径

It is the radius of the sphere.

位置

It is the position of the center of the sphere.
